# 智慧笔记隐私政策

**发布日期**：2026年03月06日  
**最新更新**：2026年03月06日  

智慧笔记（以下简称"我们"）深知个人信息对您的重要性，并会尽全力保护您的个人信息安全可靠。我们致力于维持您对我们的信任，恪守以下原则，保护您的个人信息：权责一致原则、目的明确原则、选择同意原则、最少够用原则、确保安全原则、主体参与原则、公开透明原则等。

请您在使用我们的产品和/或服务前，仔细阅读并充分理解本政策。如您不同意本政策的任意内容，您应立即停止使用我们的产品和/或服务。当您使用我们提供的产品和/或服务时，即表示您已同意我们按照本隐私政策收集、使用、存储、共享、转让和披露您的相关信息。

---

## 一、我们如何收集和使用您的个人信息

### 1.1 信息收集原则

我们遵循"最小够用"原则，仅收集为提供服务所必需的信息。

### 1.2 信息类型与收集方式

| 信息类型 | 收集目的 | 收集方式 | 是否必需 |
|---------|---------|---------|---------|
| **微信openid** | 用户唯一识别 | 自动采集 | ✅ 必需 |
| **微信昵称** | 用户个性化展示 | 用户授权 | ❌ 可选 |
| **微信头像** | 用户个性化展示 | 用户授权 | ❌ 可选 |
| **笔记内容** | 提供AI回复服务 | 用户主动输入 | ✅ 必需 |
| **设备信息** | 数据统计、性能优化 | 自动采集 | ❌ 可选 |
| **使用行为** | 产品改进、数据分析 | 自动采集 | ❌ 可选 |

### 1.3 信息收集详细说明

#### 1.3.1 用户身份信息

**微信openid**
- **收集目的**：唯一识别用户身份，确保数据安全隔离
- **收集方式**：通过微信云开发自动获取
- **是否必需**：是，不提供无法使用服务
- **存储期限**：账号存续期间
- **删除方式**：账号注销后删除

**微信昵称/头像**
- **收集目的**：个性化展示，提升用户体验
- **收集方式**：用户主动选择提供
- **是否必需**：否，提供默认值
- **默认值**：
  - 默认头像：`/images/avatar.png`
  - 默认昵称："投资爱好者"
- **存储期限**：账号存续期间
- **删除方式**：用户可随时删除或修改
- **撤回同意**：用户可在个人中心随时删除或修改

#### 1.3.2 用户内容信息

**笔记内容**
- **收集目的**：提供AI回复服务，存储用户笔记
- **收集方式**：用户主动输入
- **是否必需**：是，不提供无法使用核心功能
- **存储期限**：账号存续期间
- **删除方式**：用户可随时删除笔记
- **数据归属**：用户创作的笔记内容归用户所有

#### 1.3.3 设备与使用信息

**设备信息**
- **收集目的**：数据统计、性能优化、问题排查
- **收集方式**：自动采集
- **是否必需**：否
- **包含内容**：
  - 设备型号
  - 操作系统版本
  - 微信版本
  - 网络类型
- **存储期限**：180天
- **删除方式**：定期自动删除

**使用行为信息**
- **收集目的**：产品改进、用户体验优化、功能优化
- **收集方式**：自动采集
- **是否必需**：否
- **包含内容**：
  - 功能使用频率
  - 使用时长
  - 操作路径
  - 错误日志
- **存储期限**：180天
- **删除方式**：定期自动删除

### 1.4 信息使用方式

我们使用收集的信息用于以下目的：

1. **提供核心服务**
   - 笔记创建、保存、编辑、查看
   - AI回复服务
   - 数据存储与同步

2. **优化用户体验**
   - 个性化展示
   - 功能优化
   - 性能提升

3. **安全保障**
   - 用户身份验证
   - 数据安全隔离
   - 防止滥用

4. **数据分析**
   - 产品改进
   - 功能优化
   - 用户行为分析

5. **法律合规**
   - 遵守法律法规
   - 配合监管要求
   - 争议处理

### 1.5 第三方服务使用

我们使用以下第三方服务：

| 第三方 | 用途 | 信息提供 |
|--------|------|---------|
| **微信云开发** | 数据存储、云函数 | openid、用户内容 |
| **DeepSeek API** | AI回复服务 | 笔记内容 |
| **微信内容安全API** | 内容安全检测 | 文本内容 |

**DeepSeek API使用说明**：
- 仅发送笔记内容用于生成AI回复
- 不存储用户笔记内容
- 遵守DeepSeek API使用协议
- 最小化传输数据

## 三、我们如何共享、转让、公开披露您的个人信息

### 3.1 信息共享原则

我们不会向其他任何公司、组织和个人共享您的个人信息，但以下情况除外：

1. **在获取明确同意的情况下共享**：获得您的明确同意后，我们会与其他方共享您的个人信息。

2. **我们可能会根据法律法规规定，或按政府主管部门的强制性要求，对外共享您的个人信息**。

3. **与我们的关联公司共享**：我们的某些服务由关联公司提供，我们可能会与关联公司共享您的个人信息。

4. **与授权合作伙伴共享**：仅为实现本政策中声明的目的，我们的某些服务由授权合作伙伴提供。我们可能会与合作伙伴共享您的某些个人信息，以提供更好的客户服务和用户体验。

### 3.2 当前信息共享情况

| 共享对象 | 共享信息 | 共享目的 | 共享方式 |
|---------|---------|---------|---------|
| **微信云开发** | openid、用户内容 | 数据存储、云函数 | 必需共享 |
| **DeepSeek API** | 笔记内容 | AI回复服务 | 临时传输，不存储 |
| **微信内容安全API** | 文本内容 | 内容安全检测 | 临时传输 |

### 3.3 信息转让

我们不会将您的个人信息转让给任何公司、组织和个人，但以下情况除外：

1. **在获取明确同意的情况下转让**：获得您的明确同意后，我们会向其他方转让您的个人信息。

2. **在我们发生合并、收购或资产转让时**：我们会要求新的持有您个人信息的公司、组织继续受本政策的约束，否则我们将要求该公司、组织重新征求您的授权同意。

### 3.4 信息公开披露

我们仅会在以下情况下，公开披露您的个人信息：

1. **获得您的明确同意**。

2. **基于法律法规、法律程序、政府要求或强制性行政要求**：我们可能会根据法律法规、法律程序、政府要求或强制性行政要求，公开披露您的个人信息。

3. **基于公共利益**：为维护公共利益或保护我们及我们的用户、员工的合法权益，我们可能会披露您的个人信息。

## 四、我们如何保护您的个人信息

### 4.1 数据安全措施

我们采取以下措施保护您的个人信息：

1. **数据隔离**
   - 每个用户的数据通过openid隔离
   - 用户只能访问自己的数据
   - 数据库权限设置为"仅创建者可读写"

2. **访问控制**
   - 严格的访问权限控制
   - 最小权限原则
   - 操作日志记录

3. **数据加密**
   - 传输过程加密（HTTPS）
   - 敏感数据加密存储
   - 使用微信云开发安全机制

4. **安全审计**
   - 定期安全审计
   - 漏洞扫描
   - 安全事件响应

### 4.2 数据存储

我们使用微信云开发存储您的数据，具有以下安全特性：
- 微信云开发安全机制
- 数据实时备份
- 多地容灾
- 合规存储

### 4.3 数据保留

| 数据类型 | 保留期限 | 删除方式 |
|---------|---------|---------|
| **用户身份信息** | 账号存续期间 | 账号注销后删除 |
| **用户内容信息** | 账号存续期间 | 用户主动删除或账号注销 |
| **设备信息** | 180天 | 定期自动删除 |
| **使用行为信息** | 180天 | 定期自动删除 |
| **日志信息** | 90天 | 定期自动删除 |

### 4.4 安全事件处理

如发生个人信息安全事件，我们将：
1. 立即启动应急预案
2. 采取补救措施
3. 按规定告知用户
4. 向监管部门报告

## 五、您的权利

### 5.1 用户权利清单

按照中国相关的法律、法规、标准，以及其他国家、地区的通行做法，我们保障您对自己的个人信息行使以下权利：

| 权利 | 说明 | 实现方式 |
|------|------|---------|
| **访问权** | 查看您的个人信息 | 个人中心查看 |
| **更正权** | 修改不准确的信息 | 个人中心修改 |
| **删除权** | 删除您的个人信息 | 个人中心删除或账号注销 |
| **撤回同意权** | 撤回隐私同意 | 个人中心或账号注销 |
| **注销账号权** | 注销您的账号 | 应用内提交申请 |

### 5.2 权利行使方式

#### 5.2.1 访问权

您可以在应用内访问以下信息：
- 您的个人信息（昵称、头像）
- 您的笔记内容
- 您的使用记录

#### 5.2.2 更正权

您可以在个人中心修改：
- 昵称
- 头像

#### 5.2.3 删除权

您可以删除：
- 您的笔记内容（在应用内删除）
- 您的个人信息（账号注销时删除）

#### 5.2.4 撤回同意权

您可以通过以下方式撤回同意：
- 在个人中心删除个人信息
- 注销账号
- 停止使用服务

#### 5.2.5 注销账号权

**注销流程**：
1. 在应用内提交注销申请
2. 确认注销操作
3. 等待审核（1-3个工作日）
4. 完成注销

**注销后果**：
- 您的个人信息将被删除
- 您的笔记内容可选择保留或删除
- 您将无法使用本服务

### 5.3 响应机制

我们将在收到您的请求后15个工作日内回复。如您不满意，可通过以下方式投诉：
- 邮箱：feedback@zhihuibiji.app
- App内反馈入口
