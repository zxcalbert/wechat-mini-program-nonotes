## 二、法律实体选择与注册阶段（第3-4周）

### 2.1 法律实体对比分析

| 维度 | 个体工商户 | 有限责任公司（小微企业） |
|------|-----------|---------------------|
| **注册流程** | 简单，1-3天 | 较复杂，5-10天 |
| **注册成本** | 低（¥0-500） | 中（¥1,000-3,000） |
| **税务负担** | 定额征收，简单 | 查账征收，需记账 |
| **法律责任** | 无限连带责任 | 有限责任 |
| **融资能力** | 弱 | 强 |
| **品牌形象** | 一般 | 较好 |
| **适合阶段** | 早期验证 | 规模化运营 |

### 2.2 推荐方案

**阶段一（验证期）**：个体工商户
- 适用：产品验证、用户规模<10,000
- 优势：成本低、流程简单、税务简单

**阶段二（成长期）**：转为有限责任公司
- 适用：用户规模>10,000、考虑融资、品牌化运营
- 时机：上线后6-12个月

### 3.2 API成本控制策略

| 策略 | 实施方式 | 优先级 |
|------|---------|-------|
| **调用限制** | 每个用户每日AI调用上限10次 | P0 |
| **缓存机制** | 相同问题1小时内不重复调用API | P1 |
| **提示词优化** | 精简提示词，减少token消耗 | P0 |
| **降级策略** | API成本超预算时使用本地回复 | P1 |
| **配额预警** | 设置成本预警阈值（每月¥5000） | P1 |

### 3.3 云函数超时处理

**问题**：DeepSeek API调用可能超过云函数默认超时时间（60秒）

**解决方案**：
1. **异步处理**：云函数立即返回，后台异步调用AI
2. **状态更新**：AI生成完成后通过微信通知用户
3. **超时配置**：将云函数超时时间调整为120秒（最大）

### 4.4 内容安全合规

#### 4.4.1 微信内容安全API接入

```javascript
// 调用微信文本内容安全API
async function checkContent(text) {
  const cloud = require('wx-server-sdk');
  cloud.init();
  const result = await cloud.openapi.security.msgSecCheck({
    content: text
  });
  return result.errCode === 0; // 0表示安全
}
```
