# 每日寄信次数限制修改报告

## 一、修改概述

### 1.1 修改背景
用户需要针对同一问题向6位大师分别发送笔记，因此需要将每日寄信次数从2次调整为6次。

### 1.2 修改范围
| 范围 | 文件 | 修改内容 |
|-----|------|---------|
| 前端限制逻辑 | miniprogram/pages/write/write.js | 修改 checkDailyLimit 中的次数限制 |
| UI提示 | miniprogram/pages/write/write.js | 修改提示文案 |

---

## 二、具体修改

### 2.1 checkDailyLimit 函数修改

**原代码**：
```javascript
const remaining = Math.max(0, 2 - totalSent);
return { canSend: true, remaining: 2, total: 0 };
```

**新代码**：
```javascript
const remaining = Math.max(0, 6 - totalSent);
return { canSend: true, remaining: 6, total: 0 };
```

### 2.2 UI提示修改

**原提示**：
- "今天已寄信2次，大师需要时间深思熟虑，明天再来吧"

**新提示**：
- "今天已寄信6次，大师需要时间深思熟虑，明天再来吧"
