## 三、接口设计

调用方式：

```javascript
wx.cloud.callFunction({
  name: 'replyToLetter',
  data: {
    type: 'structure_analysis',
    analysisType: 'company', // 或 product
    content: '用户输入的公司/产品分析材料'
  }
});
```

返回结构：
- `title`
- `analysisType`
- `dimensions`
- `report`
