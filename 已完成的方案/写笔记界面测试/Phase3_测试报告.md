### 4.2 防幻觉约束内容

```
【重要约束 - 防止幻觉】
1. 只基于${mentorName}的公开言论和已知观点进行回复
2. 不要编造${mentorName}没说过的话或没做过的事
3. 如不确定某个具体观点，使用更通用的投资原则表述
4. 不要引用不存在的书籍、演讲或事件
5. 回复必须直接、具体、有针对性，避免空泛
```

### 4.3 AI免责声明格式

```
---

*以上内容为AI模拟${mentorName}的回复，仅供参考和启发，不代表该人物的真实观点或建议。*
```
