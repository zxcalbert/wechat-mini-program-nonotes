### 2.4 主题管理逻辑

```javascript
App({
  globalData: {
    themeMode: 'system'
  },

  initTheme: function () {
    const savedTheme = wx.getStorageSync('themeMode');
    if (savedTheme) {
      this.globalData.themeMode = savedTheme;
    }
  },

  setTheme: function (mode) {
    this.globalData.themeMode = mode;
    wx.setStorageSync('themeMode', mode);
  },

  getTheme: function () {
    return this.globalData.themeMode;
  },

  getThemeClass: function () {
    const mode = this.globalData.themeMode;
    if (mode === 'light') {
      return 'theme-light';
    } else if (mode === 'dark') {
      return 'theme-dark';
    }
    return '';
  },

  toggleTheme: function () {
    const current = this.globalData.themeMode;
    let next;
    
    if (current === 'system') {
      next = 'light';
    } else if (current === 'light') {
      next = 'dark';
    } else {
      next = 'system';
    }
    
    this.setTheme(next);
    return next;
  }
});
```

### 2.5 页面主题支持

**实现方式**：
1. 在每个页面的根 view 上添加 `class="{{themeClass}}"`
2. 所有样式使用 CSS 变量：`var(--变量名, 默认值)`
3. 页面加载时调用 `app.getThemeClass()` 获取主题类

**已支持的页面**：
- ✅ index（首页）
- ✅ login（登录页）
- ✅ write（写笔记页）
- ✅ detail（详情页）
- ✅ stamps（邮票页）

## 三、功能特性

### 3.1 支持的主题模式

| 模式 | 说明 |
|-----|------|
| 系统跟随 | 自动跟随系统主题设置（默认） |
| 强制亮色 | 始终使用亮色主题 |
| 强制暗色 | 始终使用暗色主题 |

### 3.2 主题切换流程

1. 用户在首页点击主题切换按钮（🌙/☀️）
2. 调用 `app.toggleTheme()` 循环切换模式
3. 模式顺序：系统 → 亮色 → 暗色 → 系统
4. 主题设置保存到本地存储
5. 页面重新渲染应用新主题

### 3.3 色彩过渡动画

```css
page {
  transition: background-color 0.3s ease, color 0.3s ease;
}
```

效果：主题切换时有 0.3 秒的平滑色彩过渡
