## 4. 独立 App 技术选型

**结论：短期不开发，推荐 uni-app 作为远期技术储备。**

| 框架 | 适配度 | 理由 |
|------|--------|------|
| **uni-app** | ⭐⭐⭐⭐⭐ | 小程序+App 一套代码，复用率最高 |
| Flutter | ⭐⭐⭐ | 高性能但需全部重写 |
| React Native | ⭐⭐⭐ | React 生态但需重写 |

**推荐路径**：小程序 + API → 如需 App → uni-app 复用 API 层
