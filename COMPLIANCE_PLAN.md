## 0. 核心结论

> **智慧笔记必须改造，但方案A（"方法论分析工具"）可行。**

### 法规适用的关键判断

《人工智能拟人化互动服务管理暂行办法》第二条明确规定：

> **提供智能客服、知识问答、工作助手、学习教育、科学研究等服务，不涉及持续性的情感互动的，不适用本办法。**

当前版本（导师人格化写信 + 首字母"我"的拟人回复）**适用本办法**。改造后（去人格化、方法论导向的分析工具）**不予适用**。

### 🔴 P1：核心功能改造（第2-4天）

#### 3.1 方案A: 分析方法分析（首选）

改造思路：将每位导师转换为一种**分析框架/方法论**，AI以该框架的视角分析用户问题，**不模拟人格**。

**提示词改造（核心）：**

**改造前（拟人化）—— 必须废弃：**
```
你是一位人生导师，你的身份是{mentorName}。请以第一人称回复...
```

**改造后（方法论工具）—— 一刀切替换模板：**
```
你是一个分析工具。请运用{mentorName}的{thinkingMethod}分析框架，客观分析以下问题。

要求：
1. 不要模拟任何人物的身份、语气或人格
2. 不要使用第一人称（"我"、"我们认为"）
3. 以书面分析报告的形式呈现
4. 列出分析逻辑和推导过程
5. 明确标注哪些是{mentorName}的核心理论，哪些是分析推导的扩展

用户问题：{content}
```

### 🟡 P2：通用生成式AI合规改造（第4-5天）

#### 4.1 AI生成标识（第十八条要求）

> **拟人化互动服务提供者应当履行人工智能生成合成内容标识义务，采取有效措施提示用户正在与人工智能服务而非自然人进行互动。**

每个AI生成内容页面（detail、roundtableResult、incubatorResult、structureAnalysisResult）：
- ✅ 右上角或顶部添加灰色小字 **"AI 生成"** 标识
- ✅ 使用 `wx:if` 条件渲染，固定在分析结果展示区域

#### 4.2 统一免责声明

在所有AI功能页面的底部添加：

```
本内容由人工智能生成，基于{分析方法名称}分析框架，仅供参考学习使用。
AI生成内容不代表任何特定个人或机构的观点，不构成任何专业建议。
```

#### 4.3 使用时长提醒（第十八条第二款）

> **对用户连续使用拟人化互动服务每超过2个小时的，应当以对话或者弹窗等方式提醒用户注意使用时长。**

在 `app.js` 中添加全局计时器：
```javascript
// 每次页面显示时检查累计使用时长
checkUsageTime() {
  const startTime = wx.getStorageSync('sessionStartTime');
  if (startTime && (Date.now() - startTime > 7200000)) {
    wx.showModal({
      title: '使用提醒',
      content: '您已连续使用2小时，建议适当休息',
      showCancel: false
    });
    wx.setStorageSync('sessionStartTime', Date.now());
  }
}
```

#### 4.4 退出途径（第十九条要求）

在侧边菜单或设置中提供"一键退出AI分析模式"按钮，点击后：
- 停止当前分析进程
- 清空当前页面状态
- 返回首页

#### 4.5 内容过滤机制

| 层级 | 措施 | 位置 |
|------|------|------|
| 第一层 | DeepSeek API自带内容安全 | replyToLetter云函数 |

> **拟人化互动服务提供者应当向用户提供交互数据复制、删除等选项。**

- ✅ 一键删除所有AI生成历史（已在profile页面已有删除功能，需确认是否包括roundtable/incubator/structureAnalysis数据）

## 8. 风险评估

### ✅ 低风险（改造后）
- 产品定位调整为"方法论分析工具"，明确不适用《拟人化互动服务管理暂行办法》
- 核心分析方法能力100%保留（21种分析框架、圆桌、孵化器、结构分析）
- 开发量可控：核心改动集中在提示词+UI文案，逻辑不变
- 旧数据可兼容展示

### ⚠️ 中风险（需持续关注）
- **算法备案**审核周期不确定（60-90工作日可能更长）
- **微信审核**标准可能随时收紧
- 如果政策进一步明确要求"工具型AI"也需要额外审批，需关注

### 🎯 最终建议

1. **坚定不移走方案A（方法论分析工具）**：这是最合规、最接近原有产品定位的转型路径
2. **同步启动算法备案**：不要等改造完再备案，周期可以很长
3. **保留历史数据的同时防御性显示**：旧数据不做大规模清理，但标记为"历史记录"
4. **先改代码后改审核**：7月15日前完成全部改造，微信审核可在此之前或之后提交
5. **持续监控政策动向**：生成式AI监管仍在快速演进
