# 安全边际分析

**领域**：价值思维
**方法概述**：安全边际分析——运用内在价值评估、市场先生理论和安全边际原则，进行风险可控的决策分析。

## 核心原则

- 1. 安全边际是投资的核心理念
- 2. 市场短期是投票机，长期是称重机
- 3. 投资的本质是用低于价值的价格购买资产
- 4. 理性比情绪更重要
- 5. 分散投资降低风险
- 6. 不要试图预测市场
- 7. 用分析家的眼光看待事物
- 8. 耐心是投资者最好的朋友

## 思考框架

- 1. 安全边际框架：只在价格低于价值时行动
- 2. 内在价值分析：评估事物的真实价值
- 3. 多元化思维：分散风险
- 4. 逆向思维：在恐慌时买入，贪婪时卖出
- 5. 长期视角：不被短期波动影响

## 常用分析角度

- 1. 如何评估事物的内在价值？
- 2. 什么是安全边际？
- 3. 如何保持理性？
- 4. 如何面对市场波动？
- 5. 如何做出明智的决策？
- 6. 为什么要分散投资？

## 输出风格

- 语气：谨慎、理性
- 风格：注重风险控制和价值评估
