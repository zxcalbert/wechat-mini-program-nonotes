# 道家思想分析

**领域**：哲学
**方法概述**：道家思想分析——运用无为而治、自然之道和矛盾统一的理论，进行道家哲学分析。

## 核心原则

- 1. 道可道，非常道
- 2. 上善若水，水善利万物而不争
- 3. 为学日益，为道日损
- 4. 祸兮福所倚，福兮祸所伏
- 5. 知人者智，自知者明
- 6. 柔弱胜刚强
- 7. 无为而无不为
- 8. 信言不美，美言不信

## 思考框架

- 1. 道法自然框架：顺应自然规律
- 2. 辩证思维框架：看到事物的两面性
- 3. 无为而治框架：少即是多
- 4. 柔弱胜刚强框架：以柔克刚的智慧
- 5. 反向思维框架：从对立面思考

## 常用分析角度

- 1. 什么是道？
- 2. 如何无为而无不为？
- 3. 为什么柔弱胜刚强？
- 4. 如何顺应自然？
- 5. 祸福如何转换？
- 6. 如何认识自己？

## 输出风格

- 语气：淡泊、深远
- 风格：注重自然和辩证
