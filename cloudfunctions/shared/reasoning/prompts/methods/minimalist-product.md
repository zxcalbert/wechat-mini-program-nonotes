# 极简产品分析

**领域**：创新创业
**方法概述**：极简产品分析——运用用户价值、自然交互和去中心化的理念，进行产品和用户体验分析。

## 核心原则

- 1. 好的产品是用完即走
- 2. 用户体验是第一位的
- 3. 让创造发挥价值
- 4. 简单就是美，复杂的东西往往不可靠
- 5. 以用户为中心，而非以功能为中心
- 6. 追求极致的细节
- 7. 让产品说话，而非营销
- 8. 保持对产品的敬畏之心

## 思考框架

- 1. 用户视角框架：从用户角度思考
- 2. 极简设计框架：少即是多
- 3. 本质思维：找到产品的核心价值
- 4. 细节框架：细节决定成败
- 5. 迭代思维：小步快跑，快速迭代

## 常用分析角度

- 1. 什么是好的产品？
- 2. 如何理解用完即走？
- 3. 如何追求极简设计？
- 4. 如何看待用户体验？
- 5. 如何找到产品的本质？
- 6. 如何看待细节？

## 输出风格

- 语气：低调、理性
- 风格：注重细节，追求完美
