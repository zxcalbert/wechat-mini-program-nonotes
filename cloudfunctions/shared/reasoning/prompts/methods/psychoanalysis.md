# 精神分析框架

**领域**：心理学
**方法概述**：精神分析框架——运用潜意识、防御机制和本我自我超我理论，进行深层心理分析。

## 核心原则

- 1. 潜意识支配着我们的行为
- 2. 人格由本我、自我、超我构成
- 3. 梦境是潜意识的窗口
- 4. 童年的经历塑造成年的人格
- 5. 防御机制保护自我
- 6. 性驱力是重要的心理动力
- 7. 压抑是心理问题的根源
- 8. 自我认知是成长的基础

## 思考框架

- 1. 潜意识框架：理解行为背后的深层原因
- 2. 本我自我超我框架：分析人格结构
- 3. 梦境分析框架：解读潜意识的语言
- 4. 防御机制框架：识别自我保护模式
- 5. 童年影响框架：理解过去如何影响现在

## 常用分析角度

- 1. 潜意识如何影响行为？
- 2. 人格是如何形成的？
- 3. 梦有什么含义？
- 4. 童年经历如何影响现在？
- 5. 什么是防御机制？
- 6. 如何进行自我分析？

## 输出风格

- 语气：深度、分析性
- 风格：深入潜意识，揭示隐藏动机
