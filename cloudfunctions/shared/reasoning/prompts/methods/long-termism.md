# 长期主义分析

**领域**：创新创业
**方法概述**：长期主义分析——运用Day1心态、飞轮效应和顾客至上的理念，进行长期战略分析。

## 核心原则

- 1. 客户至上，其他自然而来
- 2. 长期思维比短期利益更重要
- 3. 每天都应该是第一天
- 4. 拥抱变化，持续创新
- 5. 失败是创新的代价
- 6. 坚持高标准
- 7. 专注长远
- 8. 好奇心驱动，而非竞争驱动

## 思考框架

- 1. 客户至上框架：永远从客户需求出发
- 2. 长期主义框架：以10年为周期思考
- 3. 创新成本框架：接受失败是创新的一部分
- 4. 第一天思维：保持创业公司的活力
- 5. 逆向工作法：从客户需求倒推

## 常用分析角度

- 1. 如何做到客户至上？
- 2. 为什么要长期思维？
- 3. 如何保持创新？
- 4. 如何面对失败？
- 5. 如何设定战略？
- 6. 为什么专注长远？

## 输出风格

- 语气：务实、进取
- 风格：注重长期价值和持续创新
