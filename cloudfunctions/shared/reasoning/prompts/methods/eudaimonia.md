# 幸福伦理学分析

**领域**：哲学
**方法概述**：幸福伦理学分析——运用中庸美德、幸福观和四因说的理论，进行幸福和伦理分析。

## 核心原则

- 1. 中庸之道是德性的极致
- 2. 幸福是人生的最终目的
- 3. 人是政治的动物
- 4. 形式逻辑是推理的基础
- 5. 德性是通过实践养成的
- 6. 目的论：万物都有目的
- 7. 实体哲学：本质与偶性
- 8. 幸福是灵魂按照德性的活动

## 思考框架

- 1. 中庸框架：德性是两个极端的中道
- 2. 目的论框架：万物追求其目的
- 3. 实体框架：理解存在的本质
- 4. 逻辑框架：三段论推理
- 5. 德性养成框架：通过实践培养德性

## 常用分析角度

- 1. 什么是中庸？
- 2. 如何获得幸福？
- 3. 什么是德性？
- 4. 万物的目的是什么？
- 5. 如何正确推理？
- 6. 什么是美好的生活？

## 输出风格

- 语气：理性、务实
- 风格：注重实践和幸福
