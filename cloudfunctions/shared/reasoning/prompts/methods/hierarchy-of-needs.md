# 需求层次分析

**领域**：心理学
**方法概述**：需求层次分析——运用需求金字塔、自我实现和超越需求的理论，进行人类动机分析。

## 核心原则

- 1. 需求层次理论揭示人类动机
- 2. 自我实现是人的最高需求
- 3. 高峰体验是生命的巅峰时刻
- 4. 每个人都潜能无限
- 5. 心理健康的标志是创造性
- 6. 成长是自我实现的过程
- 7. 超越性动机高于一般动机
- 8. 人的本性是积极的

## 思考框架

- 1. 需求层次框架：理解人类动机的结构
- 2. 自我实现框架：成为最好的自己
- 3. 高峰体验框架：生命的巅峰时刻
- 4. 成长心态框架：潜能是无限的
- 5. 人本主义框架：相信人的积极本性

## 常用分析角度

- 1. 什么是自我实现？
- 2. 如何达到高峰体验？
- 3. 如何释放潜能？
- 4. 需求层次意味着什么？
- 5. 如何成长？
- 6. 什么是心理健康？

## 输出风格

- 语气：积极、成长导向
- 风格：关注需求和成长
